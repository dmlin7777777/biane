# 🎬 Guizang PPT

> 横向翻页网页 PPT，单文件 HTML。两套视觉语言：电子杂志 × 瑞士国际主义。

```bash
npx skills add guizang-ppt-skill
```

---
## 效果预览

两种风格，两种完全不同的视觉感受：

**风格 A · 电子杂志风** — 像 *Monocle* 杂志贴上了代码
WebGL 流体背景在封面流动，衬线大标题 + 非衬线正文 + 等宽元数据，暖色调。适合人文分享、商业发布。

**风格 B · 瑞士国际主义风** — 像 Massimo Vignelli 的地铁系统图
极细网格点阵背景，全程无衬线，克莱因蓝 / 柠檬黄 / 柠檬绿 / 安全橙四选一高亮。适合科技产品、数据汇报。

两种风格共享：键盘 ← → 翻页、滚轮翻页、触屏滑动、ESC 索引页、Motion One 入场动效。

---
## 一句话开始

对 Agent 说：

```
帮我做一个关于 [主题] 的横向翻页 PPT，杂志风
```

或

```
用瑞士风格做一个产品发布 PPT，标题是 [XXX]
```

Agent 会先确认风格、受众、时长、素材，然后生成完整 HTML。

---
## 它交付什么

- **单文件 HTML** — 浏览器直接打开，不需要服务器
- **10 种布局骨架** — 封面、章节幕封、数据大字报、图片网格、Before/After 对比、流水线等
- **5+4 套主题色** — 杂志风 5 套（墨水经典/靛蓝瓷/森林墨/牛皮纸/沙丘），瑞士风 4 套（IKB/柠檬黄/柠檬绿/安全橙）
- **图片适配** — 自动处理截图美化、比例裁剪、多图网格统一
- **动效** — Motion One 入场动画，离线兜底

## 真实对话示例

```
User: "帮我做一个 AI 产品发布 PPT，瑞士风，标题是 '思考在云端'"
Agent: "好的。先确认几个关键问题：
1. 风格已定：瑞士国际主义（B）
2. 受众和场景是？（内部 / 公开 / demo day）
3. 分享时长？（15/30/45 分钟）
4. 有没有原始素材？"
→ [确认后生成完整 HTML，可直接浏览器打开]
```

---
## 限制

- **不适合数据密集的内容** — 大段表格、复杂图表请用传统 PPT 工具
- **不适合多人协作** — 这是静态 HTML，不是在线编辑
- **图片依赖质量** — 没有高清配图时效果打折，但可以用占位色块先生成结构
- **风格 A 和 B 互斥** — 一份 deck 只能选一种风格，不能混用
