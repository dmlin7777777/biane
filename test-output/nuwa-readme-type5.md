> **你想用芒格的方式思考投资，用费曼的方式讲清复杂概念，用塔勒布的方式应对不确定性——但没人帮你把这些思维框架变成可运行的 Agent Skill。**
>
> 女娲做一件事：把一个人的思维操作系统蒸馏成可运行的 Skill。不是复制他说过的话，是提取他怎么想。

---

## 安装

```bash
npx skills add nuwa
```

两种使用入口：

| 你知道想蒸馏谁 | 你只有模糊需求 |
|---------------|--------------|
| "蒸馏芒格" | "我想提升决策质量" |
| "做一个费曼 skill" | "有没有一种思维方式能帮我看透商业本质" |

## 选择指南

**不确定用哪个？告诉女娲你的问题，她来推荐。**

| 你的问题 | 推荐 |
|---------|------|
| "我做产品决策总是犹豫" | → zhang-yiming（算法思维）或 jobs（简约暴政） |
| "我想做独立开发但不知道怎么开始" | → naval（杠杆思维）或 paul-graham（创始人模式） |
| "我的文章没人看" | → mrbeast（注意力工程）或 x-mastery-mentor（X 写作） |
| "我想提升投资判断力" | → munger（多元模型）或 taleb（风险管理） |
| "我想把复杂的东西讲清楚" | → feynman（教学即理解） |
| "我想写有感染力的故事" | → jiangnan（江南写作技法） |

## 入选标准

每个蒸馏进女娲的思维视角必须满足：

1. **有可提取的心智模型**（≥3 个在 ≥2 个领域复现的思维模式）
2. **有足够的公开一手素材**（著作 / 长访谈 / 决策记录，非二手转述）
3. **有独特的视角**（此人给世界带来的独特认知透镜，不是通用智慧）
4. **有诚实边界**（明确什么场景下这个视角失效）

不满足的不会进。进的每一个都有 6 个维度的深度调研记录可查。

---

## 它不是

- **不是名人名言合集** — 不是"乔布斯说过 X"，是"如果用乔布斯的方式看这个问题，他会怎么拆"
- **不是角色扮演玩具** — 是思维顾问，帮你做判断，不是陪你聊天
- **不保证正确** — 每个视角都有诚实边界，标注了什么场景下这个视角失效
